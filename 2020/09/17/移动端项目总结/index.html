其他效果

页面图片不显示 （暂时没有数据的页面 css 自制）

列表页 或者 需要数据渲染的把结构写清晰 方便后端拿来直接用

能用背景图代替div元素的 用背景图

多使用和div 和 ul+li 除了数据外 尽量不使用表格 因为表格样式不好控制

插件中的一些自带的行内属性 提取出来 让代码变得整洁

任何文字可能变动的地方都要考虑到 使用overflow是多余的隐藏 避免影响 布局

减少使用flex布局 宽度固定的情况下 没必要使用

对于很多地方 可以用伪元素素来表示的 尽量使用伪元素 保证代码的结构清晰

能用图片代替的宣传和可能会换的位置 使用图片比较好 避免后期会功能更改内容 导致的样式混乱

布局尽量不要使用css3布局 对于设备会有兼容问题

pc网站要考虑到手机端的正常显示
